# LYF SOS — Native Android Emergency App

**LYF SOS** is a privacy-first, open-source **native Android emergency support application** designed to help users share their location and an optional message during real-life emergencies.

It is built with **honesty, consent, and minimalism** — not surveillance, hype, or fear.

---

## 🚨 What LYF SOS Is

LYF SOS is a **user-controlled emergency assistance tool**.

It allows a user to:
- Manually trigger an SOS.
- Capture current location.
- Add a short emergency message.
- Share this information using available communication channels.

LYF SOS does **not** claim to be an emergency service, rescue system, or law-enforcement platform.

---

## 🇮🇳 India-First Design Thinking

LYF SOS is designed with real Indian conditions in mind:

- Unstable internet connectivity.
- Low-end Android devices.
- Battery constraints.
- Privacy concerns.
- Lack of trust in intrusive apps.

The app prioritizes **reliability and dignity over automation**.

---

## 🔐 Core Principles

- **Privacy-first** — no tracking, no profiling.
- **Consent-based** — nothing happens without user action.
- **No background surveillance**.
- **No false promises**.
- **No panic-driven UI**.
- **Open source transparency**.

Emergency tools must be calm and truthful.

---

## 📱 Native Android Capabilities

Because LYF SOS is a native Android app, it can support (when enabled):

- Accurate GPS location.
- Reliable offline data storage.
- Foreground Bluetooth communication.
- Wi-Fi Direct (future scope).
- Device-level permission control.

All advanced features are **explicit, opt-in, and documented**.

---

## 🚫 What LYF SOS Will NEVER Do

LYF SOS will never:

- Track users in the background.
- Collect personal identities.
- Upload data silently.
- Run hidden services.
- Trigger SOS automatically.
- Guarantee rescue or response.
- Integrate secretly with authorities.
- Monetize user data.

If a feature compromises trust, it is rejected.

---

## 🧭 Emergency Responsibility Disclaimer

LYF SOS is **not a replacement** for official emergency services.

In India, always try to contact:
- **112** — National emergency number.
- Local police, ambulance, or fire services.

LYF SOS is a **support tool**, not a rescue guarantee.
